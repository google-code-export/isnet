Message Module Help

 

The My Message module contains all received and sent messages of the member.

 

1. Create new message: When you click on this link a new popup message dialog box appears. Enter the recipient’s name or email address, subject and the message body. Please note all the three fields are mandatory. You also have the option to add attachments of any type of files.

2. Received items The received items contains all of the incoming messages coming from another member. Click on the link to display the details of the message. You can select all messages and mark them unread or delete them. There is an option delete each item with the right cross ‘X’ button.

3. Sent items This section contains all outgoing messages.
