Vendor management

                                               

 

 

 

Navigation path: Application Management > Training Item Vendor

 

Description: This page will display all predefines vendors. An authorize user can perform the following actions on “Training Item Vendor”.

1.      Add new “Training Item Vendor”.

2.      Update Existing “Training Item Vendor”.

3.      Delete existing “Training Item Vendor”.

 

 

1 Add new Training Item Vendor.

 

 

 

Navigation path: Application Management > Training Item Vendor > Add new

 

Description: Click on “Add new” link. A new window will appear for adding new “Training Item Vendor”. Fill up necessary information and press “Add” button. System will validate information. If validation fail then system will display error messages otherwise the record will be added successfully.  Click on “Close” button if don’t want to perform any operation.

 

 

2 Update Existing Training Item Vendor.

 

 

Navigation path: Application Management > Training Item Vendor > Edit icon

 

Description: Click on edit link that you want to update. A window will appear with existing information. Edit information as you want to update and click on update button. System will validate information. If validations fail then system will display error messages otherwise information will be updated successfully. Click on close button if don’t like to perform any operation.

 

 

3 Delete existing Training Item Vendor.

 

Navigation path: Application Management > Training Item Vendor > Delete icon

Description:  Click on related Cross (delete) icon that you want to delete. System will delete the record from the system permanently.
