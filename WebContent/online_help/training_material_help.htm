Training Material

                                               

 

 

 

Navigation path: Application Management > Training Material

 

Description: This page will be list all predefines vendor name and item category in a separate drop drown. This page also displays all existing “Training Items”. User can filter “Training Items” base on vendor name and item category. An authorize user can also perform the following actions for “Training Material”.

1. Add new “Training Material”.

2. Update Existing “Training Material”.

3. Delete existing “Training Material”.

 

 

1 Add new Training Material.

 

 

 

Navigation path: Application Management > Training Material > Add new

 

Description: Select vendor name and item category from dropdown. Keep category blank if you want to create new training material category. Click on “Add new” link. A new window will appear for adding new record. Fill up necessary information and press “Add” button. System will validate information. If validation fail then system will display error messages otherwise the record will be added successfully.  Click on “Close” button if don’t want to perform any operation.

 

2 Update Existing Training Material.

 

 

 

Navigation path: Application Management > Training Material > Edit icon

 

Description: Click on edit link that you want to update. A window will appear with existing information. Edit information as you want to update and click on update button. System will validate information. If validation fail then error messages will be display otherwise the record will be updated successfully. Click on close button if don’t like to perform any operation.

 

 

3 Delete existing Training Material.

 

Navigation path: Application Management > Training Material > Delete icon

 

Description:  Click on related Cross (delete) icon that you want to delete. System will delete the record from the database permanently.
