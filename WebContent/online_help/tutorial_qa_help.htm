Tutorial Q & A

                                               

 

 

 

Navigation path: Application Management > Tutorial Q & A

 

Description: This page will be display all predefines “Tutorial Q & A”. An authorize user can filter “Tutorial Q & A” base on module name. Also user can perform the followings action for “Tutorial Q & A”.

1 Add new Question and Answer

2 Update existing question and answer.

3 Delete existing question and answer.

 

 

1 Add new Question and Answer.

 

 

 

Navigation path: Application Management > Question and Answer > Add new

 

Description: Select module name from drop down and click on “Add new” link. A new window will appear for adding new record. Fill up necessary information and press “Add” button. System will validate information. If validation fail then system will display error messages otherwise the record will be added successfully.  Click on “Close” button if don’t want to perform any operation. Don’t need to fill up question and answer fields if you want to add tutorial content. Just fill up page title and select flash slide for page audio field. Similarly don’t need to enter flash file  if you want to add tutorial questions and answer. Just need to fill up question and answer fields with page title.

 

 

2 Update Existing Question and Answer.

Navigation path: Application Management > Tutorial Q & A > Edit icon

 

Description: Click on edit link that you want to update. A window will appear with existing information. Edit information as you want to change and click on update button. System will validate information. If validation fail then system will display error messages otherwise the information will be updated successfully. Click on close button if don’t like to perform any operation.

 

 

3 Delete existing Question and Answer.

 

Navigation path: Application Management > Tutorial Q & A > Delete Icon

Description:  Click on related Cross (delete) icon that you want to delete. System will delete the record from the system permanently.
