Contact us

 

 

Navigation path: Contact us > Manage Contact Us

 

 

Description: This page will be display all existing contact base on date range selected by admin. An authorize user can perform the following action for contact.

1.   Feedback against contact.

2.      Delete existing contact.

 

 

1 Feedback against contact.

 

 

Navigation path: Contact us > Manage Contact Us > Edit icon

 

Description: Click on edit link that you want to replay. A window will appear with existing information. Fill up necessary information. System will validate information. If validation fail then system will display error messages otherwise the message will be posted successfully. Click on close button if don’t want to perform any operation.

 

 

2 Delete existing contact.

 

Navigation path: Contact us > Manage Contact Us > Delete link

 

Description:  Click on related Cross (delete) icon that you want to delete. System will delete the record from the system permanently.
