Manage Chapter

                                               

 

 

Navigation path: Mentor/Protégé Relationship Management > Manage Chapter

 

Description: This page will be display all existing chapters. An authorize user can filter Chapter record base on search criteria. An authorize user can also perform the followings action for Chapter.

1 Add new Chapter

2 Update existing chapter

3 Delete existing chapter.

 

 

1 Add new Chapter.

 

 

 

Navigation path: Mentor/Protégé Relationship Management > Manage Chapter > Add new Chapter

 

Description: Click on “Add new chapter” link. A new window will appear for adding new record. Fill up necessary information and press “Add” button. System will validate information. If validation fail then system will display error messages otherwise the record will be added successfully.  Click on “Close” button if don’t want to perform any operation.

 

 

2 Update Existing chapter.

 

 

 

Navigation path: Mentor/Protégé Relationship Management > Manage Chapter > Edit icon

 

Description: Click on edit link that you want to update. A window will appear with existing information. Edit information as you want to update and click on update button. System will validate information. If validation fail then system will display error messages otherwise the record will be updated successfully. Click on close button if don’t want to perform any operation.

 

 

11.3 Delete existing record.

 

Navigation path: Mentor/Protégé Relationship Management > Manage Chapter > Delete link

 

Description:  Click on related Cross (delete) icon that you want to delete. System will delete the record from the system permanently.
