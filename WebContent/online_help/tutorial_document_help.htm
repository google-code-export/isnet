Tutorial Document

                                               

 

 

 

Navigation path: Application Management > Tutorial Document

 

Description: This page will be display all existing documents. An authorize user can perform the followings action for “Tutorial Document”.

1 Add new Tutorial Document

2 Update Existing Tutorial Document.

3 Delete existing Tutorial Document.

 

 

 

 

1 Add new Tutorial Document.

 

 

 

Navigation path: Application Management > Tutorial Document > Add new

 

Description: Click on “Add new” link. A new window will appear for adding new record. Fill up necessary information and press “Add” button. System will validate information. If validation fail then system will display error messages otherwise the record will be added successfully.  Click on “Close” button if don’t want to perform any operation.

 

2 Update Existing Tutorial Document.

 

 

 

Navigation path: Application Management > Tutorial Document > Edit icon

 

Description: Click on edit link that you want to update. A window will appear with existing information. Edit information as you want to update and click on update button. System will validate information. If validations fail then system will display error messages otherwise the record will be updated successfully. Click on close button if don’t like to perform any operation.

 

 

3 Delete existing Tutorial Document.

 

Navigation path: Application Management > Tutorial Document > Delete icon

 

Description:  Click on related Cross (delete) icon that you like to delete. System will delete the record from the system permanently.
