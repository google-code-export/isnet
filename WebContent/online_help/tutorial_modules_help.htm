Manage Tutorial Modules

                                               

 

 

 

Navigation path: Application Management > Tutorial Modules

 

Description: This page will be display all predefines “Tutorial Documents” An authorize user can filter “Tutorial Modules” base on “Tutorial document” and “User type”. Also, user can perform the following actions for tutorial documents.

1 Add new Tutorial Modules

2 Update Existing Tutorial Modules.

3 Delete existing Tutorial Modules.

 

 

1 Add new Tutorial Modules.

 

 

 

Navigation path: Application Management > Tutorial Modules > Add new

 

Description: Select tutorial document and user type from dropdown. Click on “Add new” link. A new window will appear for adding new record. Fill up necessary information and press “Add” button. System will validate information. If validation fail then system will display error messages otherwise the record will be added successfully.  Click on “Close” button if don’t want to perform any operation.

 

 

2 Update Existing Tutorial Modules.

 

 

 

Navigation path: Application Management > Tutorial Module > Edit icon

 

Description: Click on edit link that you want to update. A window will appear with existing information. Edit information as you want to update and click on update button. System will validate information. If validation fail then error messages will be display otherwise the record will be updated successfully. Click on close button if don’t want to perform any operation.

 

 

3 Delete existing Tutorial Modules.

 

Navigation path: Application Management > Tutorial Module > Delete icon

Description:  Click on related Cross (delete) icon that you want to delete. System will delete the record from the database permanently.
