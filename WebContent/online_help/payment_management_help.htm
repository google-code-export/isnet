Payment management

 

 

Navigation path: User Management > Update Payment

 

Description: This page will be display all existing users. Authorize user can filter record base on search criteria. An authorize user can also update user payment information.

 

 

1 Update payment.

 

 

Navigation path: User Management > Update Payment > Edit icon

 

Description: Click on edit link. A new window will appear for updating payment information. Press “Update” button. System will validate information. If validations fail then system will display error messages otherwise user payment information will be updated for next one year from existing date or current date.  Click on “Close” button if don’t want to perform any operation.
