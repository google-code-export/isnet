Guest/Protégé Dashboard

Guest and Protégé are used interchangeably in the MyC2i system.

The Protégé dashboard is the default home page of the Protégé. After the successful login, the Protégé is taken to the dashboard.

The Protégé dashboard is logically divided into three sections. The left most section, the middle section and the right section.

 

1. Left section: The left section contains the navigation menu, story related sub sections. Each sub section is explained below:

1.1 Top ten viewpoints section: This section contains the top ten viewpoints approved by the administrator of the site. When you submit a viewpoint, it has to be approved by the admin; once the admin confirms the viewpoint, and it is voted upon then it is available on the dashboard if it is among the top ten voted viewpoints or the most voted (winning) viewpoint.

1.2 Winning viewpoint section: Every week there is a viewpoint competition, submitted by the protégé. The most voted viewpoint is selected and displayed in this section.

1.3 Browse more viewpoints (Viewpoint Voting): After clicking the browse more viewpoints, the user is forwarded to another page containing all the viewpoints with the ability to vote for a viewpoint. Each user can only vote for one viewpoint.

 

 

2. Middle section: The middle section contains the request from mentors. Each sub section is explained below:

2.1 Pending activities : This section contains all of the activities or requests from Mentor(s), like a meeting request, visit request etc.

2.2 Mentor hierarchy section: The section contains the Mentor name of the protégé. If the mentor of this protégé has a lead mentor then it shows all the mentors of this lead mentor.

 

3. Right section: The right section contains the What's new’ and Important link sections.

3.1 What’s new section : This section contains recent news posted by myc2i site admin.

3.2 Important links section: The section contains the list of important web sites related to Islam

3.3 Common Islamic terms: This section contains the list of common Islamic terms.
