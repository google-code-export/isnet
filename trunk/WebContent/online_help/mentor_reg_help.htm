Mentor Registration Help page

 

The mentor registration option will allow the user to register with the myc2i system as a ‘mentor’. After the successful registration an initial password will be sent to the user email address. The user has to provide some personal information to register to indentify the user. The fields are self explanatory in the registration process. ALL private information like phone, email, address, etc. are kept strictly confidential. Guests and protégés contacted by the Mentor do not see this information in the mentor profile. Information like Ethnicity, Marital Status, Year of Birth are used to find the best match in terms of a Protégé. These fields are entirely optional and may be left blank, which may result in a less satisfactory match.
