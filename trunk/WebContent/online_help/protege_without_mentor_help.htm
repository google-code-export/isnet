Protégé without mentor

 

 

Navigation path: Mentor/Protégé Relationship Management > Protégés Without Mentor

 

Description: This page will display all protégé which have no mentor assigned. User can display all protégés or filter any protégé using search criteria. An authorize user can assign any mentor for any protégé.

 

1.  Assign mentor.

 

Navigation path: Mentor/Protégé Relationship Management > Protégés Without Mentor > Assign mentor

 

Description: Click on “Assign mentor” button. A new window will appear for assigning mentor for selected protégé. Select mentor as you want to assign for the protégé  and press “Assign mentor” button. System will validate information. If validations fail then system will display error messages otherwise selected mentor will be assigned for the selected protégé successfully.  Click on “Close” button if don’t want to perform any operation.
