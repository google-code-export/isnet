Login page Help

The login page has the following options

1.      Guest Registration:  Users will navigate to Guest registration page when they clink on this link. Guest and Protégé are used interchangeably in MyC2i.org. The Guest/Protégé Registration is for non-Muslims and Muslim youth who need information on and assistance with Islam.

2.      Mentor Registration:  Users will navigate to the Mentor registration page when they click on this link. Mentor Registration is ONLY for existing Muslims who are prepared to complete the Muslim mentorship training and certification.

3.       Application login:  This option allows the user to login to the myc2i system once they provide the user id as email address and password.

4.      Forgot password:  This option allows the user to retrieve their password if they lost their password.
