Login page

The login page has the following options

1.       Guest Registration:  User will navigate to Guest registration page when they clink on this link

 

2.       Mentor Registration:  User will navigate to the Mentor registration page when they click on this link.

3.       Application login:  This option allow user to login myc2i system once they provide the user id as email address and password.

4.       Forgot password:  This option allow user to retrieve their password if they lost their password.
