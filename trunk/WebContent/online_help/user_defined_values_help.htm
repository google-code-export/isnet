User Defined Values

 

 

 

Navigation path: Application Management > User Define Values

 

Description: This page will list all predefines “User Define Values” base on category selected by user. An authorize user can perform following actions on “User Define Values”.

1   Add new “User Define Values”.

2   Update Existing “User Define Values”.

3   Delete existing “User Define Values”.

 

 

1 Add New User Defined Values

 

 

 

Navigation path: Application Management > User Define Values > Add new

 

Description: select category from dropdown if want to add new “User Define Value” for a predefine category otherwise keep it empty. Click on “Add new” link. A new window will appear for adding new record. Fill up necessary information. Click on “Add” button. System will validate information. If validation fail then system will display error messages otherwise the record will be added successfully.  Click on “Close” button if don’t want to perform any operation.

 

 

2 Update Existing User Defined Values.

 

 

Navigation path: Application Management > User Define Values > Edit icon

 

Description: Click on edit link that you want to update. A window will appear with existing information. Edit information as you want to update and click on update button. System will validate information. If validation fail then system will display error messages otherwise the record will be updated successfully. Click on close button if don’t want to perform any operation.

 

3 Delete existing User Defined Values.

 

Navigation path: Application Management > User Define Values > Delete icon

Description:  Click on related Cross (delete) icon that you want to delete. System will delete the record from the system permanently.
